# SMM Panel Project
This is a Laravel and React SMM Panel project.

## Structure
- **backend/**: Contains the Laravel application.
- **frontend/**: Contains the React application.
- **database/**: Contains migrations and database setup files.
- **admin/**: Contains admin panel setup.
- **docs/**: Documentation and configuration files.

## Getting Started
1. Clone the repository.
2. Set up .env for backend and frontend.
3. Run migrations for the backend.
4. Start the frontend and backend servers.
